Guide - common problems

The link is not working correctly

When someone says the link does not correctly, they mean the best available version link.

  1. Go to the publisher website to see what the actual article content looks like, if it is available.
  2. Check the link that we have in Unpaywall under Best OA Location -> URL
  3. If the content in both places match then there is not a problem. Ask the user for clarification.
  4. Refresh the page by clicking the Refresh button. You can see if the link to the content changes.
  5. If the content requires a subscription on the publisher website, and the link to the content is incorrect, then you should make the content closed access by clicking the Close DOI button.
  6. If the content is available on the publisher website, but the Unpaywall link is incorrect, then you should escelate the ticket to someone else.
